# Agent Skills

A collection of reusable AI skills for Product Managers and operators.

## Available Skills

### prd-enforcer
Strict PRD auditing framework that enforces:
- Quantified context
- Measurable objectives
- Clear scope
- Rollout plan
- Monitoring

## Installation

Install a specific skill:

npx skills add github.com/<your-username>/agent-skills --skill prd-enforcer
